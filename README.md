# DevTrack (Job Tracker)

DevTrack is a job-application tracking app that helps people organize applications, track progress, and stay focused during the job hunt. It’s designed to make the process clearer, faster, and less stressful.

## Purpose

- Centralize all applications in one place
- Make progress visible (stages, status, timeline)
- Reduce missed follow-ups and forgotten details

## Target Users

- Students and fresh grads applying to multiple roles
- Career shifters managing many applications
- Anyone who wants a clean, visual job-tracking workflow

## Current Features

- ✅ Frontend UI scaffolding for the app
- ✅ Auth module with:
  - `POST /auth/signup`
  - `POST /auth/signin`
- ✅ Zod-based request validation
- ✅ JWT generation
- ✅ Basic auth tests (Vitest + Supertest)
